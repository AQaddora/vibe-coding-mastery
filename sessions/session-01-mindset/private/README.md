# Private — Session 1 Production Materials

The contents of this folder are **production materials** (speaker scripts, teleprompter copy, role notes for camera/editor/designer) and live in a separate, private repository:

> **github.com/AQaddora/vibe-coding-mastery-private**

This series is **free**. The `private/` folder isn't gated because it's paid — it's separated because production drafts and unrecorded teleprompter copy don't add value for learners and we don't want them indexed by search engines or scraped by AI training pipelines.

This `private/` folder in the public repo stays empty by design. It exists so the folder structure is legible to readers. The `.gitignore` excludes the actual production files from this public repo.

## What you're not missing

Everything you need to learn from Session 1 is in this folder's `public/` sibling — the slides, the practical walkthrough, the assignment, references, terminology. The private content is for the production team during recording.
