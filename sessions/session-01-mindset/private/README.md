# Private — Session 1 Production Materials

The contents of this folder are **paid course materials** and live in a separate, private repository:

> **github.com/AQaddora/vibe-coding-mastery-private**

This `private/` folder in the public repo stays empty by design. It exists only so the folder structure is visible to readers of the public README. The `.gitignore` explicitly excludes the production-side files (scripts, teleprompter, role notes) from this repo.

## If you have access to the private repo

Clone it alongside this one:

```bash
git clone git@github.com:AQaddora/vibe-coding-mastery-private.git \
  ~/path/to/vibe-coding-mastery-private
```

The folder structure mirrors this one (`sessions/session-01-mindset/`, etc.) so cross-references in the public README stay sensible.

## If you don't have access

You're not missing anything you need to learn from Session 1. The framework, slides, practical walkthrough, and assignment are all in this folder's `public/` sibling. The private content is for the production team during recording.
