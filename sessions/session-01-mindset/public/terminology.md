# ورقة المصطلحات — الجلسة 1
## Terminology Sheet · Session 01: The AI-First Developer Mindset

**الاستخدام:** هاي الورقة لازم يكون عندها نسخة عند 4 ناس:
1. **أحمد (المُحاضِر)** — للتأكد من تطابق النطق عبر كل الجلسات
2. **المصوّر** — لمعرفة المصطلحات اللي ممكن تحتاج تأكيد
3. **المحرّر** — للـcaptions والـtitles
4. **المترجم** (للـsubtitles الإنجليزي)

**المبدأ العام:**
- المصطلحات التقنية الأساسية → **تظل إنجليزية** (Prompt, API, Function, Model, Pydantic)
- المصطلحات اللي عندها ترجمة عربية شائعة ومفهومة → **عربية** (مطوّر، شبكة، ذاكرة)
- المصطلحات اللي اخترعها أحمد أو خاصة بالمساق → **إنجليزية مع شرح أول مرة** (PURE Loop, Doctrine)

## القسم 5: العبارات الفلسطينية المُعلَّمة (Signature Phrases)

هاي العبارات اللي بتعطي الفيديو "شخصية أحمد". المترجم لازم يحافظ على معناها مش حرفيتها.

| العبارة العربية | الترجمة الإنجليزية للـsubtitles | متى تستخدم |
|---|---|---|
| **يلا** | "Let's go" / "Alright" | تغيير موضوع، بداية فعل |
| **والله** | "Honestly" / "Trust me" | تأكيد، لحظة صدق |
| **يا شباب** | "Guys" / "Folks" | مخاطبة الجمهور |
| **خلّيني أحكيلكم** | "Let me tell you" | بداية قصة |
| **خلص** | "That's it" / "Done" | إنهاء فكرة |
| **بهيك** | "Just like that" / "Simple as that" | تأكيد البساطة |
| **شو يعني** | "Meaning" / "Which means" | شرح |
| **بصراحة** | "Honestly" / "To be real" | تأكيد |
| **عظيم** | "Great" / "Fine" (sarcastic) | في سياقات معينة sarcastic |
| **خلّينا نبني** | "Let's build" | عبارة الـoutro الأيقونية لأحمد |

**ملاحظة مهمة للمترجم:** الإنجليزي subtitles ما لازم يحاول يقلّد اللهجة الفلسطينية. الترجمة لازم تكون **مهنية، طبيعية، ومفهومة لمطوّر أمريكي/أوروبي**. الحس الفلسطيني يطلع من الصوت والوجه، مش من النص المكتوب.

## القسم 6: قواعد عامة للنطق

### قاعدة 1: الأسماء الأجنبية
عند ذكر اسم منتج أو شركة أو شخص أجنبي للمرة الأولى:
- **أحمد ينطق بالعربي**، ثم **يكتب بالإنجليزي على السلايد**
- مثال: ينطق "كلود" والسلايد فيها "Claude"

### قاعدة 2: المختصرات
المختصرات تُلفظ حرف-حرف بالإنجليزي:
- API → "إيه بي آي" ✅ (مش "آبي")
- LLM → "إل إل إم" ✅
- AI → "إيه آي" ✅ (مش "إي آي" ولا "ذكاء اصطناعي")
- DoS → "دي أو إس" ✅
- VS Code → "في إس كود" ✅

### قاعدة 3: المصطلحات المترجمة
عند استخدام مصطلح مترجم لأول مرة، أحمد يقول:
> "الـsecurity — يعني الأمان"
> أو "الـsenior — يعني المهندس الكبير"

بعد أول مرة، أحمد يستخدم المصطلح الإنجليزي مباشرة بدون شرح.

### قاعدة 4: الأرقام
- الأرقام من 1-10 → بالعربي ("واحد، اثنين، تلاتة")
- الأرقام الكبيرة أو التقنية → بالإنجليزي ("max_length 120"، "60% saved")
- النسب المئوية → بالإنجليزي ("10 بيرسنت" مش "10 بالمية")
- السنوات → بالإنجليزي ("2010", "2025") أو بالعربي ("ألفين وعشرة") — أحمد بختار
