# understand — scripts

Scripts for this skill will be built in a dedicated session.

Planned helpers:

- `restate.py` — given a task description, produces a 2-layer restatement (surface + inferred)
- `ambiguity_check.py` — runs the task against common ambiguity patterns from `~/ai-doctrine.md` and surfaces matches
- `brief_export.sh` — renders an approved comprehension brief into the project's docs/

For now, the skill operates by prompting alone.
